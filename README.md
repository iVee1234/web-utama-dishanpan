**BACA**
cara up perubahan:
1. git clone
2. buat branch
> git add .
> git commit -m "pesan"
3. lakukan push branch, pastikan berada di branchmu
> git push origin nama-branch
4. ke vscode, klik tombol **new pull request**
5. patikan muncul tulisan hijau “Able to merge” di kanan atas
6. ketik pesan, lalu klik create pull request

cara ambil perubahan:
1. jgn lupa buat branch baru
2. ambil perubahan
> git pull origin nama-branch
